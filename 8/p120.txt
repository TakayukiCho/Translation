Summary:
COSPLAY COLLECTION is an cosplay event which has been growing year by year.
It is held in different countries, mainly in Asia.
this event is often Organized in conjunction with the Ministry of Foreign Affairs and has been held 20 times already in the various fascinating cities, the last cities are: Seoul, Taipei, Vladivostok, Mountain, Phnom Penh, Jeju, Sapporo, Hiroshima, Matsuyama, Fukuoka.
The event is not only cosplayer gathering place, the event devised various ideas for cosplayers.
For example, the event collaborates with Tokyo Game Show and the winners of perform-contest are officially invited to the Tokyo Game Show.
The event also invites famous cosplayers from Japan and other countries as guests.
