Summary: 
COSPLAY COLLECTION is the event based on the cosplay that increasing in the world more and more.
It is held in different countries, mainly in Asia.
Organizing an event in conjunction with a consulate provides many opportunities, the number of already held projects is about 20, all of the events are held in the various fascinating cities, the last cities are: Seoul, Taipei, Vladivostok, Mountain, Phnom Penh, Jeju, Sapporo, Hiroshima, Matsuyama, Fukuoka.
The event is not only cosplayer gathering place, the event devised various ideas for cosplayers, for example, the event collaborates with Tokyo Game Show, and  the winners of perform-contest are officially invited to the Tokyo Game Show, the event also invites famous cosplayers from Japan and abroad as guests.
